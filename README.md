# boilit
My first project idea, boilit

## Idea

Boilit is a simple web application that helps users boil eggs perfectly.
Users can enter the number of eggs and choose their preferred boiling type (soft, medium, or hard). The app then calculates the ideal boiling time and the amount of water required.

## Why I am building this

I want to build this project to practice JavaScript and full-stack development.
